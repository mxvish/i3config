# i3config
i3 config files

# Installation
```sh
git clone https://github.com/mxvish/i3config.git
cd i3config
mv config ~/.config/i3/
sudo mv i3status.conf /etc/
```

# Check difference between codes on local machine and this repository
```sh
git clone https://github.com/mxvish/i3config
cd i3config
sh check_diff.sh 
```
